!style_blue{'''The !-ValidateVentrisErrorCodeTranslation-! Json fixture allows you to validate the contents of json messages returned by Ventris Error Code Translation API calls in the web browser. 
You can use this fixture for validating error code translation between the various components that interact with Ventris.'''}


!style_red {'''Input Column Descriptions'''}

!style_green{'''URL - contains the URL for the API call for the error code translation'''}
!style_green{'''attribute- contains the tag attribute name which is one of the following'''}
!style_green{'''respCodeDec'''}
!style_green{'''respCodeHex'''}
!style_green{'''failoverBehavior'''}
!style_green{'''severity'''}

!style_red {'''Output Column Descriptions'''}

!style_green {'''getConfig - returns the json response that contains the error code translation mapping'''} 
!style_green{'''errorCodeValues - returns the specific attribute value for the error codes'''}

!********> Variables

!define ventrishost {10.253.179.85}

*************!

!style_blue{'''1. Import package - common to all wiki pages'''}

|Import          |
|com.comcast.core|
|com.comcast.apps.api.ventris.metrics|


!style_blue{'''2. The following example shows how to get the !-S1ToDsmccErrorMap-! for Back Office Code 0 on an Arris device.'''}

!|ValidateVentrisErrorCodeTranslation|
|url                            |getConfig?|
|http://${ventrishost}:8080/translate?client=3&interface=S1ToDsmccErrorMap&code=0&plant=Arris||

!|ValidateVentrisErrorCodeTranslation|
|attribute|errorCodeValues?|
|respCodeDec|0|
|respCodeHex|0x00000000|
|failoverBehavior|ABORT|
|severity|HIGH|

!style_blue{'''3. The following example shows how to get the !-S1ToDsmccErrorMap-! for Back Office Code 1 on an Arris device.'''}

!|ValidateVentrisErrorCodeTranslation|
|url                            |=getConfig?|
|http://${ventrishost}:8080/translate?client=3&interface=S1ToDsmccErrorMap&code=1&plant=Arris||


!|ValidateVentrisErrorCodeTranslation|
|attribute|errorCodeValues?|
|respCodeDec|36892|
|respCodeHex|0x0000901c|
|failoverBehavior|ABORT|
|severity|LOW|
|respErrorMsg|We're sorry, but we are experiencing an issue at the moment. Please try your selection again in a few minutes or try a different one. For more info, please visit Comcast.com/help. (SRM-901C)|
